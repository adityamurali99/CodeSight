# CodeSight AI

CodeSight is an AI-powered code reviewer that finds bugs, security flaws, and logic errors in your Python code using a cloud-hosted GPT-4o brain.

## Features

- **Instant Code Review**: Analyze your current file with a single command.
- **Deep Insights**: Identifies issues that standard linters miss.
- **AI Quick Fixes**: Don't just find bugs‚Äîfix them instantly with the built-in "Apply AI Fix" lightbulb suggestions.

## How to Use

1. Open a Python file in VS Code.
2. Press `Cmd+Shift+P` (Mac) or `Ctrl+Shift+P` (Windows).
3. Type **"CodeSight: Review Current File"** and press Enter.
4. Look for the red squiggles. Click the lightbulb üí° to apply the suggested AI fix.
